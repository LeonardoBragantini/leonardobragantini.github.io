Startup Overview

MoveSlots is a startup concept that provides an AI-driven high-density storage system for small and medium urban warehouses.

The system is designed to make warehouse storage more efficient by reducing the need for fixed aisles and by using the available space in a smarter way.

MoveSlots combines:

  • Autonomous ground robots
  • Multi-layered pallets
  • A digital inventory control interface
  • AI-based warehouse configuration and movement optimization

The goal is to help warehouses store more goods in the same space, retrieve products faster, and manage inventory with better accuracy.

Role of the Robots

The robots are the physical part of the MoveSlots system.

Their function is to move under pallets, lift them, and transport them to the correct position inside the warehouse. They can also bring a specific pallet to an exit or picking area when requested by the user through the digital system.

In the full MoveSlots concept, the robots would work together with the software platform. The AI would decide which pallet should be moved, where it should be stored, and which robot should perform the task.

Prototype Description

My prototype was built using Lego Technic components, Arduino, and 3D-printed parts that allow the different components to be integrated together.

The main components are:

  • Lego Technic frame
  • Wheels and motors for movement
  • Mobile lifting platform
  • Arduino board for control
  • Infrared receiver and remote control
  • Two ultrasonic distance sensors
  • 3D-printed connectors and supports
  • TB6612FNG motor controller
  • Three LEDs indicating the state of the robot
  • External battery pack
  • Power bank to provide stable current to the Arduino

Control Modes

Manual Mode

In manual mode, the robot is controlled using an infrared remote control. The user can move the robot forward and backward, and can also activate the lifting platform.

Autonomous Mode

In autonomous mode, the robot can move by itself under a pallet.

The robot identifies the correct position under the pallet, lifts it using the mobile platform, and moves it toward the destination selected by the user.

Pallet Detection System

To identify the position of the pallet, the prototype uses two ultrasonic sensors.

These sensors measure distance. When both sensors detect that the robot is correctly positioned under the pallet, they send a signal to the Arduino. The Arduino then activates the lifting mechanism.

This allows the robot to understand when it is aligned with the pallet and ready to lift it.

Power Supply

The robot is powered by an external battery pack for the motors and by a power bank that provides stable current to the Arduino.

Purpose of the Prototype

The prototype was created to demonstrate the basic logic behind the MoveSlots robotic system.

It shows how a small autonomous ground robot could move under a pallet, detect its position, lift it, and transport it. Even though it is a simplified version, it represents the core idea of the MoveSlots technology.
