Line Follower Robot

Autonomous robot using sensors and motor control

← Back to projects

Overview

This project is a small autonomous robot designed to follow a line on the ground. It uses sensors to detect the path and adjusts the motors accordingly.

Technologies used

  • Arduino
  • Infrared sensors
  • DC motors
  • Motor driver

What I learned

I learned how to combine sensor readings with motor control logic to create a basic autonomous behavior.
